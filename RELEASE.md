# 在 GitHub 上创建 Release

## 1. 推送标签

```bash
git push origin v0.0.1
```

## 2. 网页创建 Release

1. 打开 https://github.com/forkcc/QuickMQ/releases
2. 点击 "Draft a new release"
3. 选择 tag: v0.0.1
4. Title: v0.0.1
5. 描述可粘贴 docs/release-v0.0.1.md 内容
6. 可选：上传 target/quickmq-0.0.1.jar
7. 勾选 Set as the latest release → Publish release
